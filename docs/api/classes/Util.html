setIfInitializedAndFound

  • matchProp
  • init

)
static

Defined in lib/api/util.js:108

This method is intended to set a property on an object, only if the value that property is being set to is contained on a different object in a seperate list. Additionally, if that list is uninitialized (i.e. null or undefined not merely empty), then this method will attempt to initialize it using a supplied function. When finished, executes the callback function with a boolean representing if the value was set successfully.

Parameters:

  • obj Object

    The object whose property will be set.

  • prop String

    The name of the property being set on obj.

  • value Any

    The value prop is being set to.

  • list Any

    The list of objects, one of whose properties must equal value.

  • matchProp String

    The name of the property on the list's objects being compared to value.

  • init Function

    The initialization function for the list, must return a defined list in a Node-style callback.

  • [callback] Function optional

    The optional callback function to be called with a boolean representing whether the property was successfuly set.

Throws:

Error:

If the initialization function fails to return a defined list, or passes an error of its own.

Example:

var obj = { property: 'before' },
    prop = 'property',
    value = 'after',
    list = null,
    matchProp = 'name';
function initFn (callback) {
    list = [
        { name: 'before', other: 'someValue' },
        { name: 'during', other: 'someOtherValue' }
    ];
    callback(null, list);
}

util.setIfInitializedAndFound(obj, prop, value, list, matchProp, initFn, function(found) {
    console.log(found); // false
    console.log(obj); // { property: 'before' }
});

console.log(list !== null); // true

list.push({ name: 'after', other: 'yetAnotherValue' });

util.setIfInitializedAndFound(obj, prop, value, list, matchProp, initFn, function(found) {
    console.log(found); // true
    console.log(obj); // { property: 'after' }
});
