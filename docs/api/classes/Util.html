Validates an options object by checking for required properties. If an optional validation function is provided, this method will check against that as well.

Parameters:

  • options Object

    The options object to be validated.

  • required String[]

    A list of required property names.

  • [customValidate] Function optional

    An optional, additional validation function to check options against further criteria. Invoked with the options object.

Returns:

Error | Null:

An OptionError if one or more of the required options are missing, or the result of calling customValidate with the options object, otherwise null if no custom validation function is provided.
